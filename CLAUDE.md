Brand Guidelines:
- Modern and businesslike aesthetic
- Minimalistic look
- High contrast
- Sharp, clean images
- Desaturated color photography
- Energy and enthusiasm in messaging
- Tagline: "Walk in the Light"

REQUIREMENTS:
- Single HTML file with embedded CSS and JavaScript (no external dependencies)
- Professional presentation format suitable for client review
- Support for embedded videos (MP4), GIFs, images, and annotations
- Smooth animations and transitions throughout
- Fully responsive design
- Print-friendly option

DESIGN FEATURES:

Navigation:
- Sticky sidebar or top navigation to jump between sections
- Progress indicator showing where you are in the presentation
- "Back to top" button
- Section numbers/labels

Layout System:
- Each page section should have:
  * Large section header with page/section name
  * Description/strategy text
  * 2-3 column grid for showing multiple perspectives
  * Media (images, GIFs, videos) with captions
  * Annotation callouts pointing to specific elements
  * "Key Points" or "Notes" boxes

Media Display:
- Lightbox for full-screen image viewing
- Video player with controls
- GIF auto-play on scroll into view
- Image comparison sliders (before/after or option A vs B)
- Side-by-side layouts for showing variations

Visual Style:
- Clean, minimal design (think Apple keynote or Pitch.com aesthetic)
- Generous white space
- Professional typography
- Subtle shadow and depth
- Smooth fade-in animations as you scroll
- Color-coding for different page types
- Visual hierarchy: section > subsection > detail

Interactive Elements:
- Tabs for showing multiple layout options
- Toggle between desktop/tablet/mobile views
- Expandable sections for additional detail
- Hover tooltips for annotations
- Click to zoom on images

TECHNICAL REQUIREMENTS:
- Self-contained: all CSS and JS inline
- Optimized for modern browsers
- Smooth scrolling
- Lazy load media for performance
- Support for embedding:
  * Images (JPG, PNG, GIF, SVG)
  * Videos (MP4) with poster images
  * Animated GIFs
  * YouTube/Vimeo embed codes
- Easy to customize with clear HTML comments
- Template sections that can be duplicated

STYLE INSPIRATION:
- Combine the polish of Behance case studies
- The structure of UX presentation decks
- The interactivity of modern web experiences
- The clarity of design system documentation
