# Flare Lighting Case Study

A professional, interactive presentation showcasing the website vision for Flare Lighting.

## Brand Identity

| Element | Value |
|---------|-------|
| Primary Color | `#ed7102` (Orange) |
| Secondary Color | `#31617f` (Dark Teal) |
| Font Family | Cairo (Google Fonts) |
| Tagline | "Walk in the Light" |

## Usage

Open `index.html` in any modern browser to view the presentation.

## Brand Guidelines

- Modern and businesslike aesthetic
- Minimalistic, high-contrast design
- Desaturated color photography
- Cairo font for all text (supports English and Arabic)

For full brand guidelines, refer to `Flare Identity Guidelines.pdf`.
